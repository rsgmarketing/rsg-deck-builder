# RSG Deck Builder

Build professional, brand-consistent PowerPoint decks with Claude. You describe what you want — Claude writes the code and delivers the .pptx.

## Quick Start

1. Open **Claude Cowork** and point it at this folder
2. Tell Claude what you need:

> I need a 12-slide deck about our Fast-Trak pricing strategy for the leadership team. Norlake brand.

3. Claude will show you slide headlines for approval, then generate the deck
4. Download the .pptx, open in PowerPoint, and review
5. Ask for changes in plain English — Claude rebuilds the whole deck in seconds

## What to Tell Claude

- **What** the deck is about (topic, paste in notes/docs if you have them)
- **Who** will see it (leadership, dealers, customers, sales team)
- **Which brand** (Norlake, Master-Bilt, RSG Corporate, or Norlake Scientific)
- **What you want them to do** after seeing it (approve, buy, decide)

## After You Download

Open the .pptx in PowerPoint and check:

- Is any text cut off or wrapping oddly?
- Do the fonts look right? (If not, install **Bebas Neue** — or **Teko** for Norlake decks)
- Does anything overlap at the bottom of slides?

Tell Claude what to fix. Be specific: *"Slide 5 headline wraps to two lines — shorten it"* works better than *"fix slide 5."*

- **Default (RSG / Master-Bilt):** Bebas Neue + Segoe UI — should already be on your machine
- **Norlake decks:** Teko + Trade Gothic Next — ask IT if you don't have these, or tell Claude to use the default fonts instead
